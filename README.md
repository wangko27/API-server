
# NULS-API-SERVER
Nuls API SERVER.

## Get Start

### Start Way

`change to the root folder of the project`

* command
    >`mvn clean -DskipTests=true package`

    >`cd target`

    >`java -jar API-Server-1.0-SNAPSHOT.jar`

* run script

    * Windows
    
        double click startup.bat, then REST api is up and running.
    
    * Mac or Linux
        >`chmod +x startup.sh`
    
        >`./startup.sh`
